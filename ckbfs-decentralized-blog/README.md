Decentralized Blog Platform on Nervos CKB

Monorepo for a decentralized WordPress-style blog publishing platform built on Nervos CKB.

The project stores blog posts, metadata, and media using CKBFS to enable decentralized and censorship-resistant publishing.

Frontend: HTML / CSS (initial read-only MVP)
Storage: CKBFS
Blockchain: Nervos CKB

This project is developed as a learning and implementation exercise as part of the CKB Builder program.
